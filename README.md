# ZatPat

ZatPat is a Java-based desktop application developed using JavaFX that connects users with trusted domestic service providers such as maids, cleaners, and caterers. The application integrates with Firebase to manage real-time data and service listings.

##  Features
- View list of available maids, cleaners, and caterers
- Real-time data fetching from Firebase Firestore
- User-friendly interface using JavaFX
- Centralized navigation system for different services
- Displays service provider details dynamically

##  Technologies Used
- Java
- JavaFX
- Firebase (Firestore)
- Firebase Admin SDK

##  How the Project Works

1. The application initializes Firebase using configuration credentials.
2. Data is fetched from Firebase collections (e.g., maid list).
3. The system retrieves and displays service provider information such as images and details.
4. JavaFX is used to launch and manage the user interface.
5. Users can navigate through different service categories.
